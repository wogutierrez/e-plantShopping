🌱 e-plantShopping
An interactive React-based shopping application designed for plant lovers. This project simulates an online nursery store where users can browse plants, add them to a cart, and manage purchases in a clean, responsive interface.

🚀 Features
Plant Catalog – Browse a variety of plants with images and details.

Shopping Cart – Add, remove, and update items dynamically.

Real-time State Management – Powered by Redux for consistent cart logic.

Responsive UI – Works seamlessly across desktop and mobile.

Modern Build Tools – Developed with Vite for fast builds and previews.

Linting & Formatting – ESLint + Prettier integration for clean, maintainable code.

⚙️ Installation & Setup
Clone the repository

bash
git clone https://github.com/wogutierrez/e-plantShopping.git
cd e-plantShopping
Install dependencies

bash
npm install
Run the development server

bash
npm run dev
Build for production

bash
npm run build
Preview production build

bash
npm run preview
